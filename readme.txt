== Description ==

The plugin provides a shortcode and a template tag to embed the lates video or current live stream of a specified YouTube channel.
It comes with an options page to let you set the default options for YouTube ID, width and height. You can customize these parameters with the shortcode and template tag if needed.

== Installation ==

1. Upload the complete `ix-show-latest-youtube` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. GoTo the plugins options page and setup your default settings
6. Done
